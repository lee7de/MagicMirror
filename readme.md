# 说明
魔镜项目，大学时期尝试用树莓派制作的一个软硬件结合的项目。
# idea 
使用一面镜子实现人脸认证、情绪检测、语音交互等功能，属于是家居智能化。因为技术力，只是做到了基础信息的显示，具体实现可关键词搜索 “魔镜”、“Magic Mirror”等。
